OLS Multivariate Regression


Description

We started with 1,460 records and 80 features. Before evaluating the correlation of each feature on the sale price, we cleaned ane prepared the data in various ways including:

  • replace null values with 0, median or mean
  • label hot encodiing
  • one label hot encodiing

We evaluated the training data and testing data and had a score of 91.3%. This an accurate model relative to our objective in the beginning, but I wanted to see if I reduce the number of features needed for an acccurate prediction

I performed a recursive feature elimination and got the number of features down to 13. However this decreased the accuracy of the model to 86%. Depending on what is more important to the user, either model could be used.
