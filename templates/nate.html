Description

We started with 1,460 records and 80 features. Upon research, we noticed a majority of categorical features had null values. In order to effectively clean the model, I removed all features with null values. At this point I evaluated which numerical features had the highest correlation to SalePrice. I then reviewed the data to find that it was not normally distributed.

We used a Box-Cox transformation to normalize the data. I then randomly divided the data into a train and test set. I then used a Lasso Regression with GridSearchCV to find the best alpha and train and test the model.

I used an alpha of 0.001 and the training data had a score of 83.4% and the testing data had a score of 86.5% and a MSE of 0.145.
